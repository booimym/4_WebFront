글자 관련 태그

b 태그 : 단순히 글자를 굵게 표시하는 태그
i 태그 : 단순히 글자를 기울이는 태그
ABC(태그는 중복 적용이 가능하다)


strong 태그 : 중요성,심각성 등을 표현하는 강조 태그
em태그 : 문장 일부분의 중요성을 강조할 때 사용한다.

mark 태그: 형광펜 효과를 나타내는 태그
u태그 : under Line(밑줄)을 나타내는 태그
s태그 : 취소선
del태그 : 삭제선(없어진 텍스트)
small 태그 : 작은 글자를 표시
sup 태그(윗첨자) : x2
sub 태그(아랫첨자) : log10 abbr(abbreviation :약어) 태그 : 마우스 오버 시 툴팁 출력
HTTP
