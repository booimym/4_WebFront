이미지 관련 태그(img)


    -웹 문서에 사진이나 그림 같은 이미지를 삽입하는 용도의 태그

    -img태그는 내용이 없으므로 종료 태그가 존재하지 않음. (<><>사이에 내용을 안쓰는거임)

    ※ img 태그는 인라인 형식이다.
    

    [속성]

    1)src : 삽입할 이미지의 경로를 지정하는 속성(파일 경로, 웹 주소)
    
    2)width/height : 이미지의 크기를 지정하는 속성

    3)alt : 이미지를 설명하는 속성

        1) 이미지가 출력되지 않는 경우_설명해주는 용도
        2) 웹 접근성(시각 장애인)용_스크린 리더가 읽어준다...

src 속성

웹 주소를 이용한 이미지 출력

현재 컴퓨터에 존재하는 이미지 출력

     -절대 경로 : 하나의 절대적인 기준점을 토대로 목표하는 위치까지 이동하는 경로
    
        ex) 서울시 중구 남대문로 120 2층 B강의장
        ex) C : / D : 등의 드라이브 폴더가 기준.
        ex) C:\Users\user1\eclipse-workspace\4_WebFront\images\cats\cat1.jpg


        ex) VSCODE는 워크스페이스(열려있는 최상위폴더)가 기준이 됨
        -> 이런식으로 써야 됨 : /images/cats/cat1.jpg
        
        
        
        
        -상대 경로 : 현재 위치를 기준으로 목표하는 위치까지 이동하는 경로를
        
        ex) 옆반
        
        ex) 작성중인 파일이 기준임
        .. : 상위 폴더로 이동
        / : 하위폴더, 파일 지정
alt 속성

width / height


    - 고정 크기 단위 : 화면 크기와 관계 없이 항상 지정된 크기 값을 갖는 단위
    ex) px, pt 
    (화면 크기 줄이면, 다음 줄로 넘어가거나 이런 식으로 대응하지 사진 크기가 줄여지지 않음...)

    - 가변 크기 단위 : 화면이나 영역에 따라 크기가 변하는 단위
    ex) %, em, rem

고정 크기 단위(px)

※ 가로로는 넘어가지 않는데, 세로로는 영역을 벗어날 수 있음...

가변 크기 단위(%)
